# Snakes

[![Gitter chat](https://badges.gitter.im/opencv-Snakes/Lobby.png)](https://gitter.im/opencv-Snakes/Lobby)

A simple game of snakes implemented in OpenCV and C++.

## Requirements  
* OpenCV  
* C++ (GCC recommended)

## Compilation  
`make`

## Execution
`./snakes`
